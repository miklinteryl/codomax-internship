## ✨ Features
* **View Posts:** Dynamically fetch and display all blog posts from the server.
* **Create Posts:** Form integration to submit new blogs to the backend.
* **Inline Editing:** Update existing blog posts directly from the UI.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (Fetch API)
* **Backend:** Node.js, Express.js
* **Database:** In-memory array (Local server storage)

## 💻 How to Run Locally
To run the full application with backend functionality on your machine:
1. Clone the repository: `git clone https://github.com/miklinteryl/codomax-internship.git`
2. Navigate to the directory: `cd codomax-internship`
3. Install Express (if not already installed): `npm install express`
4. Start the server: `node server.js`
5. Open your browser and visit: `http://localhost:3000`
